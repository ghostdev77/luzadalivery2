# **App Name**: Luzia Delivery

## Core Features:

- Product Catalog: Browse products by category with filtering and sorting options, displaying product images, names, prices, and descriptions.
- Shopping Cart: Add products to a shopping cart, view cart contents, update quantities, and remove items.
- WhatsApp Order Finalization: Generate a formatted order list and redirect the user to WhatsApp with the pre-filled order message.
- Real-time product search: Allow users to search products using product names and tags, then display results in real-time.
- Product Recommendations: Suggest related products and popular items based on viewing history. This can be aided by a tool that analyzes customer shopping trends and recommends which products go well together.
- Data persistence: Store product, category and order data in Firestore, including product details, categories, and order information.

## Style Guidelines:

- Primary color: Navy Blue (#1A2B4A) for headers, footers, and main texts, providing a sense of trust and reliability.
- Accent color: Golden Yellow (#F5B841) for buttons and highlights, drawing attention to key CTAs and interactive elements.
- Secondary accent color: Red (#E63946) for promotions, badges, and alerts, indicating urgency or special offers.
- Headline font: 'Poppins', sans-serif, for titles, ensuring clarity and modern appeal.
- Body font: 'Inter', sans-serif, for body text and descriptions, providing excellent readability on various devices.
